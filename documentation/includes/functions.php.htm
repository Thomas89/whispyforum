generateHexToken
Description generateHexToken generates a random hexadecimal token in this format: 3c43:a968:3f69:3480:32fe:1206:d835 and returns it as a variable.

This function is nearly-identical to generateHexTokenNoDC, but this does group the string in 4 char blocks seperated with a double colon.

 

generateHexTokenNoDC
Description generateHexTokenNoDC generates a random hexadecimal token in this format: 3c43a9683f69348032fe1206d835 and returns it as a variable.

This function is nearly-identical to generateHexToken, but this does not group the string in 4 char blocks seperated with a double colon (NoDC).

 

saveThumbnail functions (originalImage, new_height, filename)
Description The saveThumbnail functions generates a thumbnail with a set height from an image file and saves it as a file. The input and output files' extension remain the same.

There are three function using the same code. They only differ in image types:

  • saveThumbnailJPEG - to use with JPEG files
  • saveThumbnailGIF - to use with GIF files [WARNING! GIF thumbnails become still images, not animations]
  • saveThumbnailPNG - to us with PNG files
Parameters originalImage - original image file's path and name
new_height - thumbnail's height in pixels
filename - output file path and name (without extension)
Return value Image file with set filename.
Uses GD / GD2 image library
