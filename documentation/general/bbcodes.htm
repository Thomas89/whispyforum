BB Codes

PHP-based systems usally use BB codes rather than HTML tags to enable users format their texts. BB codes look the same like HTML tags: to generate bold text, HTML uses <b>text</b> which outputs to text. In BB code, the same looks like: [b]text[/b].

The usage of BB codes rather than HTML tags is a security concern to prevent HTML injection of malicious scripts. BB codes can be tested at tool.bb-test.php.
